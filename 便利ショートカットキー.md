# 基本ショートカットキー

#### タブ復活

```
command shift t
```

#### タブを閉じる

```
command w
```

#### ブラウザ検証ツール矢印

```
command shift c
```

#### ブラウザ上タブ切り替え

```
command 切り替えたいタブの番号を数字で入力
```

#### web ページを再読み込み

```
command r
```

#### アプリケーションの終了

```
command q
```

#### 全てのアプリケーションを終了

```
command option esc
```

#### スクリーンショット

```
command shift 4
```

#### 画面収録

```
command shift 5
```

#### 取り消しの取り消し

```
command shift z
```

<br>

# vscode

#### プレビュー

```
command shift v
```

#### 文の途中から改行

```
command enter
```

#### コピー

```
option shift ↓
```

#### コードのフォーマット

```
option shift f
```

#### 行の入れ替え

```
option 矢印上下
```

# ターミナル

#### 新しいタブを追加

```
command d
```

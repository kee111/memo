# 便利なショートカットキー

## 基本
#### タブを閉じる
```
command w
```
#### webページを再読み込み  
```
command r
```
<br>

## vscode
#### プレビュー
```
command shift v
```
#### 便利な改行  
```
command enter
```
#### コピー  
```
option shift ↓
```
#### コードのフォーマット
```
option shift f
```
#### 行の入れ替え
```
option 矢印上下
```
<br>

## ターミナル
#### 新しいタブを追加
```
command d
```
